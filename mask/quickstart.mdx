### Install [@reactour/mask](https://www.npmjs.com/package/@reactour/mask)

<CodeGroup>

```bash npm
npm i -S @reactour/mask
```

```bash yarn
yarn add @reactour/mask
```

</CodeGroup>

### Usage

```js
import { Mask } from "@reactour/mask";

function App() {
  const sizes = {
    width: 100,
    height: 100,
    top: 100,
    left: 100,
  };

  return (
    <>
      {/* ... */}
      <Mask sizes={sizes} />
    </>
  );
}
```
